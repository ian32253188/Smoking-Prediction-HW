# Smoking‑Prediction‑HW

## 專案簡介  
本專案為「吸菸預測」模型練習，旨在透過資料處理、特徵工程與機器學習／深度學習方法，預測或分類吸菸行為或相關指標（依據資料而定）。  
你可以將其作為作業、比賽或教學示範用途。

## 目錄結構  
```
Smoking‑Prediction‑HW/  
├── 二元化.py              ← 資料二元化處理腳本  
├── 競賽資料/              ← 原始或預處理資料檔案（例如比賽提供資料集）  
├── 提交記錄與程式/        ← 過程程式碼與版本記錄  
└── README.md              ← 本檔案  
```

## 執行環境與需求  
- Python 版本：建議 Python 3.8 或以上  
- 主要套件（依需求可補充）  
  - `numpy`  
  - `pandas`  
  - `scikit‑learn`  
  - `matplotlib`、`seaborn`（若有視覺化）  
  - 其他可能用到的深度學習框架（如 `tensorflow`、`pytorch`）  
- 建議使用虛擬環境（如 venv、conda）管理環境  
- 若要複現程式，請先在命令列執行：  
  ```bash
  pip install -r requirements.txt
  ```  
  （如你已有 `requirements.txt` 檔案）

## 資料說明  
- 原始資料位於 `競賽資料/` 資料夾  
- 請先執行 `二元化.py`，以完成資料的二元化處理（例如：吸菸／非吸菸、是／否等標籤）  
- 資料處理流程包括：缺值處理、特徵轉換、標籤編碼、資料分割訓練集／測試集等  
- 特徵與標籤細節請參考程式碼與註解

## 模型訓練流程  
1. 載入處理後資料  
2. 特徵工程（如 one‑hot encoding、標準化／正規化）  
3. 模型選擇與訓練（如邏輯回歸、隨機森林、SVM、深度神經網路）  
4. 模型評估：使用準確率、精確度、召回率、F1 分數、ROC‑AUC 等指標  
5. 儲存模型（如使用 `joblib`、`pickle`）以便後續推論

## 如何執行  
在專案根目錄下執行以下命令：  
```bash
python 二元化.py        # 完成資料預處理  
python train_model.py   # 訓練模型（假設你有此脚本）  
python predict.py       # 使用訓練完成的模型進行預測（假設你有此脚本）  
```
（若檔案名稱不同，請依實際名稱調整）
